  Step 1 — Create components/Leaderboard.tsx (new file).

  Props (define a Leaderboard generic-typed component):
  - `rows: T[]` — opaque list of row data (the page passes its own row objects).
  - `columns: Array<{ key: string, header: string, render: (row: T) => ReactNode, mobileSlot?: 'primary' | 'secondary' | 'meta' | 'hidden', widthLg?: string }>` — column descriptors.
    - `mobileSlot`: at base, columns map to slots inside the card layout: `primary` = big metric, `secondary` = stacked label-value pairs, `meta` = footer line, `hidden` = not rendered at base.
    - `widthLg`: optional Tailwind grid-template-column value used at lg: only (e.g., `60px`, `1fr`, `100px`).
  - `mobileLayout: 'card'` (only supported value for v1; future-proofing).
  - `getRowKey: (row: T) => string` — required for React keys.
  - Optional: `rankRender?: (row: T, index: number) => ReactNode`.

  Rendering:
  - At lg: render a CSS-grid table:
    ```
    <div className="hidden lg:grid lg:gap-2" style={{ gridTemplateColumns: columns.map(c => c.widthLg ?? '1fr').join(' ') }}>
      {/* header row */}
      {columns.map(c => <div key={c.key} className="text-xs font-mono uppercase text-terminal-text-dim px-2 py-1">{c.header}</div>)}
      {/* data rows */}
      {rows.map(r => columns.map(c => <div key={c.key} className="px-2 py-2 font-mono text-sm text-terminal-text">{c.render(r)}</div>))}
    </div>
    ```
    Style binding via inline `style={{ gridTemplateColumns }}` is acceptable here because column widths are dynamic per usage (Tailwind cannot express arbitrary user widths).
  - At base (lg:hidden): render a card stack:
    ```
    <div data-leaderboard-mobile className="flex flex-col gap-3 lg:hidden">
      {rows.map((r, i) => (
        <div key={getRowKey(r)} className="border border-terminal-border bg-black/40 p-3 flex flex-col gap-2">
          {/* rank if provided */}
          {/* primary slot: big number */}
          {/* secondary slots: label-value pairs */}
          {/* meta slot: footer */}
        </div>
      ))}
    </div>
    ```
    Note: add `data-leaderboard-mobile` attribute on the wrapper div for Playwright test instrumentation (per plan 06 Task 2 Step 3).

    Inside the card:
    - Rank (top-left) + primary slot (top-right, big): `flex items-baseline justify-between`.
      - Rank: `text-terminal-text-dim font-mono text-xs`.
      - Primary: `font-mono font-bold text-2xl text-terminal-green` (the "BOUGHT & PAID FOR" big number).
    - Secondary slots: rendered as `flex flex-col gap-1`, each row `flex justify-between gap-2 text-xs font-mono` with label `text-terminal-text-dim uppercase` and value `text-terminal-text break-words`.
    - Meta slot: `text-xs font-mono text-terminal-text-dim border-t border-terminal-border/50 pt-1`.

  All Tailwind only — no inline styles other than the dynamic gridTemplateColumns.
